✅ 간단한 추가/조회 기능 구현

근로자 및 업체 정보를 메모리에서 관리 가능

[3주차] CRUD 기능 구현 (추가, 수정, 삭제, 조회)

✅ 간단한 콘솔 UI 개선

근로자/업체 정보를 추가, 수정, 삭제할 수 있음
조회 기능을 통해 특정 정보를 검색 가능

✅ 파일 데이터 불러오기 기능 (loadWorkers(), loadCompanies())
출력 목표:

근로자 및 업체 정보가 파일에 저장되고, 프로그램을 재실행해도 유지됨

✅ 입력값 검증 (잘못된 입력 방지, 예외 처리)
✅ UI 개선 (메뉴 선택 방식 개선, switch-case 활용)
